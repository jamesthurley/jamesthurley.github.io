Off The Wall is a small web workbench for Kilter Board owners who want to inspect, test, and play with a training board without starting a full climbing session.

It can connect to supported Kilter/Aurora controllers from a compatible browser, show what the board is advertising over Bluetooth, keep the selected board layout visible, draw colors onto holds, import images, build simple animations, and send frames to the LEDs.

The hosted web app runs entirely in your browser. There is no account system and no Off The Wall cloud service.

A native iOS app is coming soon. It will use the same local-first Off The Wall app, with iOS-native Bluetooth and file sharing support.
