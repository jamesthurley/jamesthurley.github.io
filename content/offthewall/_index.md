Off The Wall is a small app for Kilter Board owners who want to have fun on with the wall between climbs.

It can connect to supported Kilter/Aurora controllers from a compatible browser or the iOS app, and lets you draw, create animations,
and even play games like Pong on the Kilter Board.

The hosted web app runs entirely in your browser. There is no account system and no Off The Wall cloud service.

A native iOS app is coming soon. It will use the same local-first Off The Wall app, with iOS-native Bluetooth and file sharing support.

[Launch Off The Wall Web App](/offthewall/app/)

## Limitations

This app has not been tested on all Kilter Board configurations and hardware.
Board configurations which have not been tested are marked as "Unverified".
If you do manage to test the app on an Unverified configuration [let me know](/offthewall/support)
if it works or not so that I can remove the "Unverified" flag.

The Safari browser does not support Web Bluetooth, so you will have to use the iOS App instead if you are on an Apple device.
